Scratch Juniors

Perfect for young beginners, this course introduces core computing ideas using playful, visual programming with Scratch and hands-on learning with micro:bit devices. Students build animations, interactive stories, and simple games while learning sequencing, loops, events, and creative problem-solving.

Course Tools

  • Scratch Programming: Visual block-based coding at scratch.mit.edu
  • micro:bit: Physical computing with BBC micro:bit microcontrollers
  • Scratch for micro:bit: Connecting Scratch projects to real hardware
  • AI Learning: Introduction to artificial intelligence through hands-on activities

Syllabus for Ages 6-10

Term 1: Getting Started

  • Introduction to Scratch interface and sprites
  • Basic movement commands (move, turn, glide)
  • Creating simple animations and stories
  • Introduction to micro:bit - what is it and how it works
  • First micro:bit project - making it display your name
  • AI Basics: What is AI? How computers can "think" and learn
  • Simple AI concepts through Scratch - making characters that respond to you

Term 2: Interactive Projects

  • Events and user interaction (when clicked, when key pressed)
  • Creating simple games (maze games, catch the apple)
  • micro:bit sensors - buttons, accelerometer, compass
  • Connecting Scratch to micro:bit for interactive projects
  • Building a digital pet or simple robot controller
  • AI in Games: Creating characters that make decisions and learn from players
  • Simple pattern recognition - teaching the computer to recognize shapes or sounds

Term 3: Creative Coding

  • Loops and repetition (forever, repeat blocks)
  • Variables and simple counting
  • Creating music and sound effects
  • micro:bit radio communication between devices
  • AI Creativity: Building chatbots and virtual assistants in Scratch
  • Machine learning basics - teaching computers to recognize patterns in data
  • AI ethics for kids - understanding how AI helps and how to use it responsibly
  • Final project showcase - interactive story or game with AI elements

Learning Outcomes

  • Understand basic programming concepts through visual coding
  • Create interactive stories, games, and animations
  • Use micro:bit sensors and outputs effectively
  • Develop logical thinking and problem-solving skills
  • Build confidence in technology and creativity
  • Learn to share and collaborate on projects
  • AI Understanding: Grasp basic concepts of artificial intelligence and machine learning
  • Create simple AI-powered projects and understand how computers can "learn"
  • Develop responsible AI awareness and ethical thinking about technology

Recommended ages: 6–10. No prior experience required. All projects are designed to be fun, engaging, and age-appropriate for primary school children.
